This example demonstrates the clean integration pattern:

  • Primary: Fetches from data/anime.json
  • Fallback: Uses cdn/latest-anime.json if primary fails
  • Error Handling: Shows demo data if both fail
  • Simple Play: Direct anime playback with one click
